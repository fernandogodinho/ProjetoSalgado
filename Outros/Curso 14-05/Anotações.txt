Instalaçao do ttp-server
npm install -g http-server

Subir o servidor do node js

Ir até a pasta onde deseja subir o servidor "http-server", escrever "http-server" e vai subir o servidor.
Para selecionar a porta que deseja subir, é só utilizar o "http-server -p3000", irá subir o http-server na porta 3000.
Por default irá subir na porta 8080.
